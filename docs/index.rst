iTerm2 API Wrapper
=================

``iterm2-api-wrapper`` is a Python package that wraps parts of the iTerm2 Python API
and provides a small CLI for interacting with iTerm2 windows, tabs, sessions, and
UI dialogs.

.. note::

   This project targets **macOS** and requires **iTerm2** with the Python API enabled.
